# Отель динозаврика

Браузерная мини-игра: заселяйте добрых животных и прогоняйте монстров.

## Играть онлайн

**https://sashasosnova.github.io/dinosaur-hotel/**

Открывается на любом устройстве в браузере (телефон, планшет, компьютер).

## Локально

Откройте `index.html` двойным кликом — сборка и зависимости не нужны.
